Hotel Night Spotter

A modern web application designed to help users discover hotels, night spots, and local places with a clean and intuitive interface. The project is built with a fast, scalable frontend stack to support smooth development and deployment.

Project Overview

This application is structured for easy development, quick updates, and efficient performance. You can edit and run the project in any development environment you prefer.

How to Edit and Run This Project
Using Your Local Machine (Recommended)

Make sure you have the following installed:

Node.js

npm

Setup Steps

git clone https://github.com/SantoshRM-1/hotel-night-spotter.git
cd hotel-night-spotter


Install dependencies
npm install

Start the development server
npm run dev

This will start the app locally with auto-reload.

Editing Files Directly in GitHub

Open the file in your GitHub repo

Click the Edit (pencil) icon

Make changes

Commit the update
